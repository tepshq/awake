迷わない、シンプルなパネル

状態と切り替えだけ。ごちゃごちゃしない。

  • クリックで ON / OFF（切替時だけパスワード）
  • オフ強制までの時間を 1 / 3 / 6 / 12h で設定
  • リポジトリへもワンクリック

付けっぱなしを、ちゃんと止める

設定時間を超えると、オフにするまで消えないダイアログが出ます。AIを回しっぱなしの「鞄の中で発熱」を防止。

AWAKE

なぜ AWAKE か

🟢

見える化

メニューバーに 👀 / 👁️ で常時表示。切り忘れに気づける。

🔒

安全設計

sudoers 改変も常駐ヘルパーも無し。切替時だけ標準のパスワード。

⏰

必ずオフ

長時間で「オフにするまで消えない」ダイアログ。付けっぱ事故を防止。

🚀

導入がラク

ターミナルに1行。SwiftBar 導入から配置まで自動。

インストール

ターミナルにコピペして実行するだけ。

curl -fsSL https://raw.githubusercontent.com/tepshq/awake/main/install.sh | bash

SwiftBar 未導入でも自動で入ります（Homebrew が必要）。
